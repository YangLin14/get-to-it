Get To It

不只是待辦清單。是一個很懂你、會記得你、會在你走偏的時候把你拉回來的 AI 私人秘書。

「為想法太多、容易分心、但真的想前進的人設計」
01

這是什麼？

Get To It 是一個安裝在 Claude 桌面 app 裡的 AI Skill。它的角色不是「清單管理工具」，而是一個私人秘書——它知道你所有的目標、專案、任務和想法，會在每天早上告訴你今天最重要的三件事，在你被新想法帶偏的時候提醒你，在你停滯太久的時候認真跟你說。

🧠
動力感知優先排序
不把人當機器。當你動力最高的時候，系統知道。當你今天只有 3 小時，它不會給你 5 小時的任務。
🤵‍♀️
有情緒智商的秘書
平時溫柔支持。你跳過任務給了理由，它會記住。但拖了三天，它會認真跟你說話。
💡
想法捕捉 + 評估
隨手丟入想法，AI 自動評估跟你目前目標的關聯度，決定要加入主線、放進想法庫、還是下週再看。
🤝
人機協作清單
任務可以分配給你或 AI Agent 執行。清單統一追蹤，Agent 完成後自動更新進度。
📈
動量視覺化
不只看百分比。看你最近幾天的節奏，是在加速還是在減速。距離目標的感覺，比數字更真實。
🧩
14 天滾動記憶
你說「今天頭痛先不碰這個」，明天它不會再用同樣力道推你。它記得你的狀態。
02

安裝方式

Get To It 是一個 Claude Cowork Skill，需要在 Claude 桌面 app 裡安裝。

三步驟完成安裝

確保你已安裝 Claude 桌面 App 並開啟 Cowork 模式

1️⃣ 下載 get-to-it.skill 檔案
2️⃣ 在 Claude 桌面 App 中 點擊 .skill 檔案 → 「Save Skill」
3️⃣ 開新對話，說 「早安」 就啟動了
第一次使用：系統會自動初始化資料庫，儲存在你選擇的工作資料夾裡（.get-to-it.db）。這個資料庫在每次對話之間持續存在，你的目標和進度不會消失。

秘書人格系統

這是 Get To It 最特別的地方之一。秘書不是固定語氣，它會根據你最近的狀態動態調整：

😊
日常模式 — 你有在穩定前進
觸發條件：今天或昨天有完成任務
「昨天你完成了三個任務，動力不錯！今天我幫你挑了……」溫柔、支持、慶祝小進步。
🌤
溫柔提醒 — 停滯 1-2 天
觸發條件：1-2 天沒有任何任務完成
「昨天好像沒有動到什麼，沒事的。不過有幾件事在等你……今天哪怕只花 20 分鐘推一件，都是前進。」
⚡
認真施壓 — 停滯 3 天以上
觸發條件：連續 3 天以上沒有實質進展
「Yang，我要認真跟你說了。你已經四天沒有進展了。今天不需要做完什麼大事，但你必須動一下。告訴我能做的最小那一步是什麼？」
記憶加持：如果你昨天說「頭痛先不碰這個」，今天即使是施壓模式，秘書也不會推這個任務。它記得你說過的話。

過人之處——跟其他工具的差異

市面上的 Motion、Notion、ClickUp 等工具都假設你是有紀律的人，只需要更好的組織方式。Get To It 針對的是不同的人——

★ 市場唯一
動力感知優先排序
所有工具都假設你每天都是「就緒狀態」。Get To It 把你當前的動力和可用時間列為優先排序的核心因素。新想法出現的那一刻動力最高——它知道。
★ 情緒護城河
溫柔但施壓的秘書人格
Duolingo 貓頭鷹的生產力版本。有個性的提醒比冷冰冰的通知有效十倍。沒有任何生產力工具在做這件事。
★ 業界首創
人機協作清單
任務可以在你和 AI Agent 之間流動。你做決策，Agent 執行，清單統一追蹤。Taskade 有 AI agents，但它是團隊工具，不是個人秘書。
★ 防分心核心
想法庫 + 關聯度評估
新想法出現時，現有工具要麼讓你忽略它，要麼讓你被帶走。Get To It 第三條路：立即評估、智慧分類、延後處理，動力保留，主線不斷。
★ 越用越懂你
14 天滾動記憶
你給的每個跳過原因、每個情緒標記，系統都記得 14 天。一個不記得你昨天說什麼的秘書，不是好秘書。
★ 動量思維
節奏視覺化而非百分比
「你完成了 60%」沒什麼感覺。「你最近 7 天：🔥🔥😐🔥🔥😴🔥，動量指數 72，趨勢加速中」——這個你感覺得到。

一句話說清楚：

其他工具幫助「已經有紀律的人更有效率」。
Get To It 幫助「有動力但容易走偏的人找到方向並堅持走下去」。

MVP 使用小技巧

🎯 第一次用：從一個目標開始

不要一次把所有目標都輸入進去。挑一個你現在最在乎的目標，讓秘書幫你拆解成具體任務，用一週看看。

⏱ 每天說「今天有幾小時」

這一個動作讓優先排序算法有時間感知，推給你的任務才會真的適合今天。

🔥 收工前記得回報 emoji

🔥 / 😐 / 😴 這三個心情回報是動量追蹤的核心。持續兩週，系統就能畫出你真實的工作節奏曲線。

💡 有想法就直接說

不需要先判斷「這個值不值得記錄」。直接說，秘書幫你判斷。讓你的腦子只負責想，不負責整理。

📝 跳過任務時一定給原因

「跳過，因為今天沒時間」這個原因存入記憶後，秘書明天會幫你考慮時間因素。沒有原因的跳過，系統只能猜。

Telegram / OpenClaw 整合

Get To It 的晨間簡報不只能在 Claude 桌面 app 看到——你可以讓它主動推送到你的 Telegram，透過 OpenClaw 實現真正的「讓秘書追你」。

❓ 能直接從 ClawHub（clawhub.ai）安裝嗎？

不行，兩個系統格式不同。

Get To It 是一個 Claude Cowork Skill——它存在於 Claude 桌面 App 裡，用 Claude 作為 AI 引擎，透過 SKILL.md 規則控制 AI 行為。

ClawHub / OpenClaw 是另一套獨立的 local AI 平台，有自己的 Skill 格式、自己的 Gateway 協定和 AI runtime，主要功能是連接 20+ 通訊平台（Telegram、WhatsApp、Slack 等）並在你的設備上本地執行 AI。

兩個平台的 .skill 格式不互通。你無法把 Get To It 的 .skill 檔案安裝到 OpenClaw，反之亦然。

✅ 正確的整合方式：CLI Pipe

Get To It 提供 morning --format telegram 指令，輸出 Telegram MarkdownV2 格式的純文字。這個文字可以用任何方式送到你的 Telegram——包括透過 OpenClaw 的 connector，或直接用 Telegram Bot API。

方式一：最簡單 — 直接用 Telegram Bot API（無需 OpenClaw）
# 1. 先設定環境變數
export GTI_DB_DIR=~/.get-to-it
export TG_BOT_TOKEN="你的 Bot Token"
export TG_CHAT_ID="你的 Chat ID"

# 2. 每天早上執行這段
MSG=$(python gti.py morning --format telegram)
curl -s -X POST "https://api.telegram.org/bot$TG_BOT_TOKEN/sendMessage" \
  -d chat_id="$TG_CHAT_ID" \
  -d parse_mode="MarkdownV2" \
  -d text="$MSG"

# 3. 加入 cron 每天早上 8:30 自動送
30 8 * * 1-5 cd /path/to/gti && bash above_script.sh

⚠️ 需要先建立 Telegram Bot（@BotFather）並取得 Token 和 Chat ID。

方式二：透過 OpenClaw（已安裝 OpenClaw 的用戶）

OpenClaw 提供 Telegram connector，你可以建立一個 OpenClaw skill 或 cron job，定期呼叫 gti.py 並將輸出透過 OpenClaw 的 connector 送到 Telegram。

# OpenClaw cron skill 範例（概念）
python gti.py morning --format telegram | openclaw send --channel telegram --chat YOUR_CHAT

具體 OpenClaw API 語法請參考 OpenClaw 官方文件。

方式三（未來規劃）— HTTP Webhook Server

未來版本計畫在 gti.py 中加入 serve --port 8080 模式，開啟輕量 HTTP 伺服器，讓 OpenClaw 或任何 webhook 工具可以直接輪詢取得晨間資料，無需 shell 整合。

# 尚未實作
python gti.py serve --port 8080
GET http://localhost:8080/morning?format=telegram
推薦做法（現在就能用）：方式一最簡單，5 分鐘可完成設定。有在用 OpenClaw 的話，方式二更優雅——OpenClaw 幫你管理 Telegram 連線，不需要自己處理 Bot API。

V2 使用小技巧

🏃 先計時再說「做完了」

說「開始做任務 #N」才能啟動自動計時。直接說「做完了」系統只能接受手動分鐘數。一個月後你會看到每類任務的真實時間分佈。

🧠 把重要限制說出來

「我每週三下午要接小孩」「這個月預算很緊」「我討厭整理 Excel」——這些說了，秘書就永遠記住，不需要每次提醒。

📱 接 Telegram 後不用每天開 Claude

連結 Telegram 之後，早上 8:30 晨間簡報就自動送達你的手機。你只需要在想深入討論任務或捕捉想法時才打開桌面 app。

🤖 不用擔心 Agent 任務卡住

交出去的任務系統會監控。超過 30 分鐘沒動靜，下次早間簡報就會告訴你，直接說「重試」或「我自己來」。
