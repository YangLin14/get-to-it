想法評估引擎 MVP
新想法輸入後自動判斷：

· 與現有目標關聯度（0-100%）
· 是否應立即處理
· 建議行動（進入主線 / 想法庫 / 下週評估）
· 動力是否足夠值得現在衝
★ 分心控制核心

動量追蹤器 MVP

主觀：每日 emoji 自評 🔥 / 😐 / 😴

· 加速 / 減速趨勢偵測
